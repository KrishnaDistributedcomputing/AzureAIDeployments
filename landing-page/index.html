Pattern 1 — Centralized Hub AI
Overview

A single shared Azure OpenAI instance behind API Management serves the entire organization. Centralized governance, cost control, and monitoring.

Key Services
  • Azure OpenAI Service
  • API Management (Standard v2)
  • Key Vault, Azure Monitor, Log Analytics
  • Entra ID for RBAC
When to Use
  • Org-wide GenAI rollout with unified policy
  • Cost-conscious deployments with shared capacity
  • Teams that don't need model customization
Cost Breakdown (Azure Live Prices)
Azure OpenAI GPT-4o (500M tokens)~$2,750
APIM Standard v2$700
AI Search S1$245.28
Key Vault + Monitor~$28
Total~$3,723/mo

Composite SLA 99.72%

Service Limits
  • AOAI: 30 deployments/resource, 240K TPM (GPT-4o)
  • APIM: 4,000 req/sec (Standard v2)

Pattern 2 — Decentralized Spoke AI
Overview

Each business unit owns its own Azure OpenAI, Cognitive Services, and networking resources within a dedicated spoke VNET.

Key Services
  • Per-team Azure OpenAI
  • Spoke VNETs with peering
  • Azure Firewall, NSGs
  • Dedicated Key Vaults per spoke
When to Use
  • Teams needing full autonomy over AI resources
  • Different compliance / data residency per BU
  • Large enterprises with independent P&Ls
Cost Breakdown (Azure Live Prices, 3 spokes)

Pattern 3 — Hybrid Central-Spoke
Overview

Central APIM gateway + shared AOAI with spoke-level fine-tuned models. Balances governance with flexibility.

Key Services
  • APIM as central gateway
  • Hub AOAI + spoke ML endpoints
  • Azure Monitor + Log Analytics
  • Hub-spoke VNET topology
When to Use

Pattern 5 — Serverless AI
Overview

Event-driven AI processing with Azure Functions, Logic Apps, and consumption-based pricing. Zero idle cost.

Key Services
  • Azure Functions (Consumption / Flex)
  • Logic Apps, Event Grid
  • Azure OpenAI (pay-per-token)
  • Cosmos DB serverless
When to Use
  • Bursty or unpredictable workloads
  • Startups / cost-sensitive deployments
  • Document processing, chatbots, event pipelines

Pattern 6 — Secure Private AI
Overview

Zero-trust AI deployment with private endpoints, confidential computing, CMK encryption, and full network isolation.

Key Services
  • Private Link / Private Endpoints
  • Confidential VMs (DCsv3)
  • Azure Firewall Premium
  • Customer-Managed Keys, DDoS Protection
When to Use
  • Healthcare, finance, government workloads
  • Data that must never traverse public internet
  • Compliance: HIPAA, SOC 2, FedRAMP
