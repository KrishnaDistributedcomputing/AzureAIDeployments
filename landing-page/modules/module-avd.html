Azure Virtual Desktop

Securely stream full Windows desktops and apps from Azure with centralized identity, policy, and autoscaling.

Overview

Azure Virtual Desktop (AVD) is Microsoft’s desktop and app virtualization platform for Windows 11/10 Enterprise multi-session and RemoteApp delivery in Azure.

It supports pooled and personal host pools, Microsoft Entra-based access controls, and cost optimization with autoscaling and start-on-connect.

Key Capabilities

  • Pooled and Personal Host Pools: Match user experience to workload needs
  • RemoteApp Delivery: Publish only apps instead of full desktops
  • FSLogix Profiles: Fast, roaming profile containers
  • Start VM on Connect: Reduce compute cost for variable usage
  • Conditional Access: Enforce MFA and compliance checks
  • Centralized Observability: Azure Monitor + Log Analytics insights

Use Cases

Secure Contractor Access

Provide isolated desktops for third-party users without exposing internal networks.

Task Worker Desktops

Run pooled sessions for call center, operations, and front-line teams.

Legacy App Virtualization

Publish specific Windows apps through RemoteApp while keeping data centralized.

Pricing Guidance

Pooled Host Pool

Typical: $120-$220 per active user/month (shared compute).

Personal Host Pool

Typical: $220-$450 per user/month (dedicated compute).
