Azure Firewall

Stateful, managed network firewall with threat intelligence and built-in DDoS protection

Overview

Azure Firewall is a managed, cloud-based network security service that protects your VNet resources. It's a stateful firewall with built-in high availability and unrestricted scalability.

Provides Layer 4 and Layer 7 filtering, threat intelligence integration, and seamless Azure Monitor integration.

Key Capabilities

  • Stateful Inspection: Full Layer 4 protocol inspection and filtering
  • Application Rules: Layer 7 filtering with FQDNs and HTTP/HTTPS inspection
  • Network Rules: Layer 4 filtering with protocols, ports, and IP addresses
  • NAT Rules: Source and destination network address translation
  • Threat Intelligence: Blocking known malicious IPs and domains
  • High Availability: Zone-redundant deployments across zones
  • Scalability: Handles unlimited connections simultaneously
  • Logging & Monitoring: Full integration with Azure Monitor and Log Analytics

Use Cases

FAQ

Can I use Azure Firewall with NSGs?

Yes. NSGs filter traffic at NIC level, Firewall filters at VNet level. Both are complementary.

What's the throughput?

Up to 30 Gbps per deployment automatically, scales as needed.

Does it support IDPS?

Yes. Intrusion Detection/Prevention System is available in Premium tier.

What about DNS filtering?

Azure Firewall can filter egress DNS traffic and log all DNS queries.
