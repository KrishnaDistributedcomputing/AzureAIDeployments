Azure SQL Managed Instance

Fully managed SQL Server instance in the cloud with high compatibility

Overview

Azure SQL Managed Instance is a fully managed SQL Server database engine with more than 99% compatibility with SQL Server. It combines the ease of fully managed service with the power and reach of SQL Server.

Run existing SQL Server workloads in Azure with minimal code changes and benefit from built-in high availability.

Key Capabilities

  • SQL Server Compatibility: 99%+ compatible with enterprise SQL Server
  • Built-in High Availability: Automatic failover and data redundancy
  • Automated Backups: Continuous backup with point-in-time restore
  • Advanced Features: Polybase, Machine Learning Services, Graph Database
  • Security: Advanced Data Security, Transparent Data Encryption, Always Encrypted
  • Instance Pools: Reduce costs by sharing resources across instances
  • Network Isolation: Private IP in VNet, no public endpoint required
  • Monitoring: Built-in Azure Monitor and Intelligent Insights integration

FAQ

Can I run all SQL Server features?

99%+ compatibility. Some features like SQL Agent and encryption scope have differences.

How do I connect to SQL Managed Instance?

Through private IP in VNet. Express Route or VPN needed from on-premises.

What's the backup retention?

7-35 days automatic backup retention. Long-term retention available up to 10 years.

Can I scale resources dynamically?

Yes. Vertical scaling available with downtime. Horizontal scaling through read replicas.
