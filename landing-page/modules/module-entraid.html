Azure Entra ID

Enterprise-grade identity and access management platform

Overview

Azure Entra ID (formerly Azure AD) is Microsoft's cloud-based identity and access management service. It enables organizations to manage user identities and access to both cloud and on-premises applications.

Entra ID provides single sign-on, multi-factor authentication, and comprehensive identity security.

Key Capabilities

  • Single Sign-On (SSO): One credential for all applications
  • Multi-Factor Authentication: Passwordless sign-in and MFA
  • Managed Identities: Service principals without managing credentials
  • Conditional Access: Risk-based access control policies
  • Identity Protection: Detect and respond to identity risks
  • Role-Based Access Control: Fine-grained permissions management
  • Hybrid Identity: Sync on-premises AD with cloud
  • B2B & B2C: External user collaboration and consumer identity

Use Cases

🔐 Secure Access

Implement zero-trust access with conditional policies and MFA.

🌐 Hybrid Cloud

Sync identity between on-premises Active Directory and cloud.

🤝 External Collaboration

Manage access for partners, vendors, and customers securely.

📱 Passwordless Authentication

Use Windows Hello, FIDO2 security keys, or Authenticator app.

🛡️ Identity Risk Management

Detect anomalous sign-ins and remediate threats automatically.

FAQ

What's the difference between Entra ID and Active Directory?

AD is on-premises directory service, Entra ID is cloud-based identity platform with additional security features.

Can I use passwordless authentication?

Yes. Multiple options: Windows Hello, FIDO2 keys, Microsoft Authenticator app.

How do I manage B2B access?

Use Entra ID B2B collaboration to invite external users and grant selective access.

What's Conditional Access?

Policy engine that evaluates conditions (location, device, risk) and enforces access decisions.
